Kami bisa membantu anda cara membeli item pilihan

  • Cari items sesuai dengan nama game yang dimainkan, ketika sudah ingin membeli item klik Tambahkan setelah itu item akan masuk ke kerajang. Ketika selesai membeli, anda harus Klik icon kerajang untuk melanjutkan pembelian. Disana anda dapat Menambah / Mengurangi item yang dibeli, ketika sudah yakin membeli item lalu anda klik checkout. Di halaman checkout anda diminta untuk mengisikan Nama & Email gunanya untuk ketika ada kendala pihak kami bisa mencek proses pembelian anda, dan anda wajib mengisikan ID maupun Server game. Apabila di game Tidak ada Server game boleh dikosongkan. Setelah mengisi klik Lanjutkan transaksi anda bisa memilih metode pembayaran. Kalau sudah selesai pembayaran tinggal Tunggu beberapa menit untuk proses pengiriman items ke game anda.

  • Masuk dengan email dan password aplikasi Paypal anda, disana anda diminta dengan pembayaran utama. Dalam pembayaran utama anda (pilih salah satu) bisa menggunakan Saldo Paypal, Kartu Bank anda, setelah itu klik Pembayaran. Kalau sudah selesai pembayaran tinggal Tunggu beberapa menit untuk proses pengiriman items ke game anda.

  • Bagian bawah terdapat 3 melalui via Instagram, Gmail, WhatsApp. Format untuk menghubungi :

    • Nama
    • Email
    • Keperluan

    1. Terjadi kesalahan pembelian
    2. Salah dalam pengisian data
    3. Pembayaran masih belum masuk
    4. Kekurangan dalam pengisian data
    5. Ketidak sesuaian antara data dan pembayaran

  • Anda tidak bisa membeli items yang berbeda game dalam 1 waktu, jadi harus bergantian dalam proses pembelian. Membedakan game terdapat pada samping nama items (singkatan dari nama game tersebut), dan dapat dilihat dari sampul items game tersebut.

  • Pengisian Id & Server tiap game boleh tulis huruf, angka, maupun symbol (sesuai dengan id dalam game). Apabila server game tidak ada maka boleh dikosongkan tidak perlu diisi lagi.
